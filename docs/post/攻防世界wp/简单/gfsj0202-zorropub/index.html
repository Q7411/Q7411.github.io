这里做了两件事 第一件事 生成 30 个随机数 每个都是

rand() % 1000

然后把这些数字的十进制字符串拼起来算 MD5 比如随机数是

12 345 6

那么 MD5 算的是

"123456"

不是数组 也不是加分隔符 最后要求 MD5 必须等于

5eba99aff105c9ff6a1a913e343fec67

def glibc_rand(seed, n):
    if seed == 0:

    r = [0] * (344 + n)
    r[0] = seed

    for i in range(1, 31):
        r[i] = (16807 * r[i - 1]) % 2147483647

        r[i] = r[i - 31]

    return [r[i] >> 1 for i in range(344, 344 + n)]
